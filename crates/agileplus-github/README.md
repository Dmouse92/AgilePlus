# agileplus-github

GitHub REST client, GitHub-to-domain mapping, and repository sync support.

## Public API Index

- Modules: `client`, `map`, `sync`.
- Mapping API includes GitHub issue/PR structs and conversion helpers.
- Sync API includes `GhDataSource`, `LiveGhDataSource`, `SyncReport`, and `sync_repository`.

## Validation

```bash
cargo test -p agileplus-github
```
