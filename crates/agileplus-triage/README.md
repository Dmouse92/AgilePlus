# agileplus-triage

Rule-based triage engine and backlog adapter for synced items.

## Public API Index

- Engine API: `classify`, `SyncedItem`, `TriageOutcome`, `TriageRule`, `TriageRules`.
- Store API: `BacklogStoreOps`.
- Modules: `adapter`, `backlog`, `classifier`, `engine`, `router`.

## Validation

```bash
cargo test -p agileplus-triage
```
