# agileplus-fixtures

Deterministic test data builders and seed payloads shared across tests and dashboard seeding.

## Public API Index

- Builders: `FeatureBuilder`, `WorkPackageBuilder`.
- Seed helpers: `seed_dogfood_features`, `seed_test_data`, `TestFixtures`.
- Payload helpers: `feature_create_payload`, `plane_webhook_payload`, `transition_payload`.

## Validation

```bash
cargo test -p agileplus-fixtures
```
