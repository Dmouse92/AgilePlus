# agileplus-telemetry

OpenTelemetry and tracing initialization for AgilePlus services.

## Public API Index

- `init_telemetry`, `TelemetryAdapter`, `TelemetryError`, `TelemetryGuard`.
- `TelemetryConfig`, `SubscriberGuard`, `init_subscriber`.
- Modules: `adapter`, `config`, `logs`, `metrics`, `traces`, plus re-exported logging helpers.

## Validation

```bash
cargo test -p agileplus-telemetry
```
