# agileplus-p2p

Peer discovery and event replication for brokerless or hybrid AgilePlus sync.

## Public API Index

- Core types: `P2pNode`, `P2pBehaviour`, `PeerId`, `P2pError`.
- Modules: `device`, `discovery`, `export`, `import`, `replication`, `vector_clock`, `git_merge`, `error`.

## Validation

```bash
cargo test -p agileplus-p2p
```
