# agileplus-api

Axum HTTP server library for AgilePlus REST and OpenAPI surfaces.

## Public API Index

- `create_router`: builds the Axum router.
- `AppState`: shared HTTP application state.
- Modules: `api_key`, `error`, `middleware`, `openapi`, `responses`, `router`, `routes`, `state`.

## Validation

```bash
cargo test -p agileplus-api
```
