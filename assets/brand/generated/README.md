# Generated Brand Icon Assets

This directory is the default output target for `tools/build-icons/Build-Icons.ps1`.

Generated files are derived from `assets/brand/source/icon.svg`. That SVG is a
placeholder by design. Do not ship generated assets as final AgilePlus brand art
until the source SVG is replaced with the approved icon design.

```powershell
pwsh tools/build-icons/Build-Icons.ps1
```
