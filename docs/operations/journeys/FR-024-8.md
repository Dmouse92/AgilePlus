# Journey: SCHEMA.md versioning

> **Status: stub** — eco-024 FR-6.

## User story
As a **schema author**, I bump the schema version in `SCHEMA.md` and the validator enforces the new shape.

## Steps
1. Edit `SCHEMA.md` to v2.
2. Bump `schema_version` in the validator.
3. Re-run on the corpus.

## Acceptance
- Schema changes are explicit and versioned.
- Validator refuses traces with a stale `schema_version`.
