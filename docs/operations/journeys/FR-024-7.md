# Journey: --check-anchors mode

> **Status: stub** — eco-024 FR-6.

## User story
As a **trace author**, I run `trace-validator --check-anchors` and it fails on any `spec_anchor` that does not match a real heading in the named `spec_slug`'s `spec.md`.

## Steps
1. Add a deliberately dangling anchor to a test trace.
2. Run validator.
3. Confirm it fails.
4. Fix the anchor.
5. Confirm exit 0.

## Acceptance
- Validator's --check-anchors mode catches dangling anchors.
