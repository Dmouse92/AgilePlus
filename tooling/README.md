# AgilePlus Tooling

Supporting tools and utilities for the AgilePlus ecosystem.

- **browser-agent-mcp** -- Python MCP server for browser automation agent
- **legacy-enforcement** -- Scanner for detecting legacy/deprecated patterns
- **worklog-aggregator** -- Rust tool for aggregating cross-project worklogs

Each subdirectory has its own build and test setup. See individual READMEs within each tool directory.
